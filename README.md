# Examen 2da Fase 

Ejercicio 3 planteado en el set de Problemas 

## Planteamiento
Diseñar una página web responsiva (utilizando SASS) para una empresa que quiere 
promocionar sus refrescos, debe tener su pagina de inicio y la sección donde se pueda 
agregar, modificar, eliminar y listar los refrescos, todo esto de manera asíncrona (usando 
JQuery). Además, nos piden subir el proyecto a un repositorio de github y llevar un control 
de versiones del proyecto.

## Valoración
8 puntos
